# ToDo-List Web-App
A simple and responsive To-Do List web-application that helps users manage daily tasks efficiently.

## Features 
- Add tasks
- Delete tasks
- Mark tasks as completed
- Responsive design
- Stores tasks locally

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open in VS Code
3. Run using Live Server
